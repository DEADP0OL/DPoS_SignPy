# DPoS_SignPy
Offline tools for signing DPoS items in Python

## Sign Items

- Messages
- Transactions (In-Development)

## Installation

```git clone https://github.com/DEADP0OL/DPoS_SignPy```

```apt-get install python3-pip```

```pip3 install pynacl```

## Usage

```cd DPoS_SignPy```

```Python3 sign.py```

Enter message and primary passphrase when prompted.
